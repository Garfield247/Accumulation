# VSCodeInstall

## 软件源安装

1. 添加ppa源

   ```
   sudo add-apt-repository ppa:ubuntu-desktop/ubuntu-make
   ```

2. 更新软件源

   ```
   sudo apt-get update
   ```

3. 安装ubuntu-make

   ```
   sudo apt-get install ubuntu-make
   ```

   打开VSCode官网https://code.visualstudio.com/

   下载Deb包

2. 安装

   ```
   cd ~/Documents
   ```
