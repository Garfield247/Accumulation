### 1. 安装deepin-wine环境

<https://gitee.com/wszqkzqk/deepin-wine-for-ubuntu>
这是deepin-wine环境的一键安装包，因为wine依赖非常的多，所以有人就写了一个shell 把所有的依赖都整理好做成一键安装。

```shell
$ git clone https://gitee.com/wszqkzqk/deepin-wine-for-ubuntu.git
$ cd  deepin-wine-for-ubuntu 
$ ./install.sh
```
